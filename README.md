# Frontend Mentor - Social Links Profile Solution

This is my solution to the Social Links Profile challenge on Frontend Mentor. The goal of this project was to recreate a responsive social profile card using semantic HTML and modern CSS practices.

## Overview

### The Challenge

Users should be able to:

- View hover states for interactive elements
- View a responsive centered profile card layout
- Navigate social links clearly and accessibly

## What I Learned

While building this project, I practiced:

- Structuring layouts using Flexbox
- Creating reusable CSS variables with `:root`
- Improving responsive sizing with `max-width`
- Building cleaner semantic HTML structures
- Understanding spacing systems and layout relationships

One thing I especially improved on was avoiding rigid fixed sizing and thinking more about flexible layouts.

Example:

```css
.profile-card {
    width: 100%;
    max-width: 420px;
}
```

I also became more comfortable organizing CSS variables for colors, typography, and spacing.
